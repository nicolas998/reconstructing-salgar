This is a README file for the Copernicus Publications LaTeX Macro Package 
copernicus_package.zip in the version 3.6, 10 November 2014
-------------------------------------------------------------------------
It consists of several files, each with its separate copyright.
This specific archive is collected for journals published by 
Copernicus Publications (Copernicus GmbH).

Copyright (C) 2014 Copernicus GmbH

E-mail: publications@copernicus.org
URL:   	http://publications.copernicus.org



Content:
- copernicus.cls: The LaTeX2e class file designed for Copernicus Publications journals. Current Version 7.15, 16 September 2014
- copernicus.cfg: The configuration file containing journal-specific information used by the class file. Last update 7 November 2014 
- copernicus.bst: The bibliographic style file for BibTeX. Current Version 1.1, April 2007 
- natbib.sty
- pdfscreencop.sty / pdfscreen.sty
- template.tex: A LaTeX template in journal style.
